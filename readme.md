
## Get started

```bash
npm install
npm run build
```

**Set env variables (support local env file .env)**
```bash
DB_HOST
DB_PORT
DB_DATABASE
DB_USERNAME
DB_PASSWORD
APP_PORT
```

**Run migration**
```bash
npm run migration
```

**Start app**
```bash
npm run start
```
